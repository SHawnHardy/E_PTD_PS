## 简介

本项目是

部分时滞诱发 Watts-Strogatz小世界 FitzHugh-Nagumo神经元网络 产生随机多共振

的数值模拟

采用 Python + C++ 的方式

数值运算部分由C++完成

控制，数据可视化由Python处理
